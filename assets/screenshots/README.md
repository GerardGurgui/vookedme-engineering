## Standards

- Format: PNG
- Resolution: sufficient for readability at 2x on retina screens (typically 1600px wide for full-page screenshots)
- Anonymisation: no real user data, no real API keys, no production URLs
- Annotation: use red arrows or boxes to highlight the specific element being discussed
- Dark theme: terminal and IDE screenshots use a dark theme for consistency
- Naming: `kebab-case-descriptive-name.png`
